Made an E-Commerce Laravel Porject from scratch.

# Following Video at : 
https://www.youtube.com/watch?v=1Vj73iP_7vk

# Tech Stack 
**Laravel** (PHP) ,

**React** with **TypeScript**, 

**Tailwindcss** with DaisyUI, 

**FilamentPHP** (components) 

**Stripe** (online payments).
